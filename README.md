# 🚀 Core-CRUD-MERN

A full-stack CRUD (Create, Read, Update, Delete) application built with the MERN Stack. This project demonstrates complete CRUD operations, RESTful API integration, MongoDB database management, and a responsive React-based user interface. It serves as a solid foundation for learning full-stack web development and building scalable MERN applications.

## ✨ Features

* ➕ Create new records
* 📖 Read and display data
* ✏️ Update existing records
* 🗑️ Delete records
* ⚡ RESTful API with Express.js
* 🍃 MongoDB database integration
* 🎨 Responsive React frontend
* 🔄 Real-time UI updates
* 🛡️ Error handling and input validation

## 🛠️ Tech Stack

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/your-username/Core-CRUD-MERN.git
```

### Install dependencies

#### Backend

```bash
cd server
npm install
```

#### Frontend

```bash
cd ../client
npm install
```

### Configure Environment Variables

Create a `.env` file inside the `server` folder:

```env
PORT=5000
MONGODB_URI=your_mongodb_connection_string
```

### Run the application

#### Start the backend

```bash
cd server
npm start
```

#### Start the frontend

```bash
cd client
npm start
```
